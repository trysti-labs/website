## Local Development

Simply open `index.html` in a web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using VS Code Live Server extension
# Right-click index.html > Open with Live Server
```

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #0ea5e9;
    /* ... */
}
```

### Content
- Update company information in `index.html`
- Modify services in the services section
- Update statistics in the about section
- Replace placeholder email and social links

### Images
Add your own images by:
1. Creating an `images` folder
2. Adding your images
3. Updating the `src` attributes in `index.html`
